## 리눅스에서 삼바 서버  
```
sudo apt-get install samba
```  

```
sudo smbpasswd -a 계정명
```  

```
sudo vim /etc/samba/smb.conf
```  

```
[원하는이름]
comment = samba test server
path = /home/공유할폴더명
browsable = yes
create mask = 0770
directory mask = 0771
writable = yes
printable = no
```  

```
sudo service smbd restart
```  

## 윈도우에서  
네트워크 드라이버 연결(N)  
  
\\\\리눅스IP주소\원하는이름  
  
리눅스에서 설정한 계정명과 비밀번호 입력  

## 만약 안되면  
리눅스에서 해당 폴더에 권한을 줘야 함  
```
chmod 777 해당폴더
```
